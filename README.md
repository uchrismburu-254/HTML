# Bootstrap Project

Simple static site using Bootstrap styles. Files include `index.html`, `about.html`, and other pages.

Preview locally by opening `index.html` in a browser or using a simple HTTP server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
